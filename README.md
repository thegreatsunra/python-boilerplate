# python-boilerplate

> Boilerplates are like toothbrushes. Everyone has one and no one wants to use anyone else's.
> 
> This one is mine.

## Dependencies

- python
- pyenv
- poetry

## Usage

```shell
## Run main script
poetry run ./src/main.py

## Lint with black
poetry run black .

## TODO: test with pytest
```
